# Motor Test Jig Firmware

Firmware development and testing for a motor diagnostic jig.

## Features
- Reverse mode when button pressed
- Speed control (+ / - buttons)
- PWM motor drive control

## Tools
- STM32
- C language
- STM32CubeIDE
- Oscilloscope signal analysis

## Purpose

This project explores improvements to a motor testing jig used to evaluate motor behaviour under different operating conditions.
